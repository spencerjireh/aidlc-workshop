## Code-editor installed components

The following has been pre-configured in your code-editor IDE:

- [Using Kiro on the command line](https://kiro.dev/cli/)

<!-- -->

- [Amazon Q Developer Extension for VSCode](https://docs.aws.amazon.com/amazonq/latest/qdeveloper-ug/q-in-IDE.html).
- [Git](https://github.com/git-guides/).

A number of programming languages are pre-installed on the IDE.

> **Note:**
>
> If Kiro on the command line detects missing software, it will ask for your approval before installing the software and proceeding.
