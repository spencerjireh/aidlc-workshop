# Start Workshop

## Objective

In this section you will:

- Set up your development environment (VS Code or code-editor).
- Connect to Kiro using an AWS Builder ID.
- Get familiar with Kiro.

## Preparing for the workshop

Follow the setup instructions in this section to prepare your environment for the workshop.

- If you are attending an AWS guided event, setup your environment [here](/event/dashboard/en-US/workshop/10-start-workshop/11-aws-event).
- If you are *not* participating in an AWS guided event, setup your environment [here](/event/dashboard/en-US/workshop/10-start-workshop/13-outside-aws-event).

> **Note:**
>
> If you are running this outside an AWS event by authenticating with Builder ID on the free tier, you will use Kiro for free and without an AWS account. You will not create any resources, and there will no charges.

## Estimated run time

15 minutes.
